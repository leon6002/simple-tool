# 彩票工具快速开始

## 快速访问

访问地址：`http://localhost:3000/tools/lottery`

## 5分钟快速上手

### 1. 生成随机号码（最简单）

1. 打开彩票工具页面
2. 点击"号码生成器"标签
3. 选择彩票类型（大乐透/双色球/福彩8）
4. 点击"生成号码"按钮
5. 点击"复制号码"按钮复制到剪贴板

**完成！** 你已经获得了一组随机彩票号码。

### 2. 计算奖金（手动输入）

1. 打开彩票工具页面
2. 点击"奖金计算"标签
3. 选择彩票类型
4. 输入中奖号码数量：
   - 前区/红球中奖个数
   - 后区/蓝球中奖个数
5. 点击"计算奖金"按钮
6. 查看中奖等级和奖金

**完成！** 你已经知道中了什么奖。

### 3. 使用 OCR 识别（高级功能）

**前提条件：** 需要配置 OCR API Key

1. 打开彩票工具页面
2. 点击"奖金计算"标签
3. 选择彩票类型
4. 点击"选择文件"按钮
5. 选择彩票照片（清晰、正面拍摄）
6. 等待识别完成
7. 查看识别的号码
8. 输入中奖数量并计算奖金

**完成！** 你已经使用 OCR 识别了彩票号码。

## 配置 OCR API（可选）

如果你想使用 OCR 识别功能，需要配置 API Key：

### 步骤 1: 获取 API Key

访问 [SiliconFlow](https://siliconflow.cn/) 注册并获取 API Key。

### 步骤 2: 配置环境变量

在项目根目录创建 `.env.local` 文件：

```env
OCR_API_KEY=your-api-key-here
OCR_BASE_URL=https://api.siliconflow.cn/v1
```

### 步骤 3: 重启服务器

```bash
npm run dev
```

**完成！** OCR 功能已启用。

## 常见问题

### Q: 生成的号码是随机的吗？
A: 是的，完全随机，每次生成都不同。

### Q: 奖金金额准确吗？
A: 固定奖金准确，浮动奖金仅供参考，以官方公布为准。

### Q: OCR 识别不准确怎么办？
A: 建议：
- 使用清晰的照片
- 正面拍摄，避免倾斜
- 确保光线充足
- 或使用手动输入功能

### Q: 支持哪些彩票类型？
A: 目前支持：
- 大乐透 (DLT)
- 双色球 (SSQ)
- 福彩8 (FC8)

### Q: 可以批量生成号码吗？
A: 目前不支持，但在未来计划中。
