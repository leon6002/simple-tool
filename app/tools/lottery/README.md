# 彩票工具 (Lottery Tools)

彩票工具提供了两个主要功能：奖金计算和号码生成器。

## 功能特性

### 1. 奖金计算

- **支持的彩票类型**：
  - 大乐透 (DLT)
  - 双色球 (SSQ)
  - 福彩8 (FC8)

- **彩票照片 OCR 识别**：
  - 上传彩票照片
  - 自动识别彩票号码
  - 支持 JPG、PNG、WebP 等格式
  - 最大文件大小：10MB

- **手动输入计算**：
  - 输入中奖号码数量
  - 自动计算奖金等级
  - 显示对应奖金金额

### 2. 号码生成器

- **随机生成彩票号码**：
  - 根据选择的彩票类型生成号码
  - 自动排序
  - 一键复制到剪贴板

## 彩票规则

### 大乐透 (DLT)
- 前区：5个号码 (1-35)
- 后区：2个号码 (1-12)
- 奖金等级：
  - 一等奖：前5+后2（浮动奖金）
  - 二等奖：前5+后1（浮动奖金）
  - 三等奖：前5+后0（10,000元）
  - 四等奖：前4+后2（3,000元）
  - 五等奖：前4+后1（300元）
  - 六等奖：前3+后2 或 前4+后0（200元）
  - 七等奖：前3+后1 或 前2+后2（100元）
  - 八等奖：前3+后0 或 前1+后2 或 前2+后1 或 前0+后2（15元）

### 双色球 (SSQ)
- 红球：6个号码 (1-33)
- 蓝球：1个号码 (1-16)
- 奖金等级：
  - 一等奖：红6+蓝1（浮动奖金）
  - 二等奖：红6+蓝0（浮动奖金）
  - 三等奖：红5+蓝1（3,000元）
  - 四等奖：红5+蓝0 或 红4+蓝1（200元）
  - 五等奖：红4+蓝0 或 红3+蓝1（10元）
  - 六等奖：红2+蓝1 或 红1+蓝1 或 红0+蓝1（5元）

### 福彩8 (FC8)
- 选择8个号码 (1-80)
- 注：福彩8的奖金计算较为复杂，暂不支持奖金计算功能

## OCR API 配置

彩票照片识别功能使用 DeepSeek-OCR API。需要在环境变量中配置：

```env
OCR_API_KEY=your-ocr-api-key-here
OCR_BASE_URL=https://api.siliconflow.cn/v1
```

## 技术实现

### 前端
- React + TypeScript
- Framer Motion (动画)
- Tailwind CSS (样式)
- React Hot Toast (提示)

### 后端 API
- Next.js API Routes
- DeepSeek-OCR API (图片识别)

### OCR 识别流程
1. 用户上传彩票照片
2. 前端将图片转换为 Base64
3. 调用 `/api/lottery/ocr` API
4. API 调用 DeepSeek-OCR 进行文字识别
5. 解析识别结果，提取彩票号码
6. 返回识别的号码给前端显示

## 使用方法

### 奖金计算
1. 选择彩票类型（大乐透/双色球/福彩8）
2. 上传彩票照片进行 OCR 识别，或手动输入中奖号码数量
3. 点击"计算奖金"按钮
4. 查看奖金等级和金额

### 号码生成
1. 选择彩票类型
2. 点击"生成号码"按钮
3. 查看生成的随机号码
4. 点击"复制号码"按钮复制到剪贴板

## 注意事项

1. OCR 识别准确率受照片质量影响，建议：
   - 照片清晰、光线充足
   - 彩票平整、无遮挡
   - 号码区域完整可见

2. 奖金金额仅供参考：
   - 浮动奖金根据当期奖池和中奖人数确定
   - 固定奖金可能因政策调整而变化
   - 请以官方公布为准

3. 号码生成器：
   - 生成的号码完全随机
   - 仅供娱乐参考
   - 理性购彩，量力而行
