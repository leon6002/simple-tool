# 彩票工具功能详解

## 功能概览

彩票工具提供两大核心功能：
1. **奖金计算器** - 计算彩票中奖等级和奖金
2. **号码生成器** - 随机生成彩票号码

## 功能 1: 奖金计算器

### 1.1 彩票类型选择

**支持的彩票类型：**

#### 大乐透 (DLT)
- **规则：** 前区5个号码(1-35) + 后区2个号码(1-12)
- **奖金等级：** 8个等级
- **最高奖金：** 一等奖（浮动）
- **最低奖金：** 八等奖（15元）

#### 双色球 (SSQ)
- **规则：** 红球6个号码(1-33) + 蓝球1个号码(1-16)
- **奖金等级：** 6个等级
- **最高奖金：** 一等奖（浮动）
- **最低奖金：** 六等奖（5元）

#### 福彩8 (FC8)
- **规则：** 选择8个号码(1-80)
- **奖金等级：** 暂不支持计算
- **说明：** 仅支持号码生成

### 1.2 OCR 照片识别

**功能特点：**
- 自动识别彩票照片上的号码
- 支持多种图片格式（JPG、PNG、WebP）
- 最大文件大小：10MB
- 实时显示识别结果

**使用步骤：**
1. 点击"选择文件"按钮
2. 选择彩票照片
3. 等待识别完成（通常 2-3 秒）
4. 查看识别的号码

**识别结果显示：**
- 主号码：灰色徽章显示
- 特殊号码：蓝色徽章显示
- 识别失败：显示错误提示

**最佳实践：**
- 使用清晰的照片
- 正面拍摄，避免倾斜
- 确保光线充足
- 号码区域完整可见

### 1.3 手动输入计算

**适用场景：**
- OCR 识别不准确
- 没有彩票照片
- 想要快速计算

**输入项：**
- 前区/红球中奖个数（0 到最大值）
- 后区/蓝球中奖个数（0 到最大值）

**计算结果：**
- 显示中奖等级
- 显示奖金金额
- 浮动奖金标注"浮动奖金"
- 未中奖显示"未中奖"

### 1.4 奖金等级详解

#### 大乐透奖金等级

| 等级 | 中奖条件 | 奖金 | 说明 |
|------|----------|------|------|
| 一等奖 | 前5+后2 | 浮动 | 最高奖金，根据奖池确定 |
| 二等奖 | 前5+后1 | 浮动 | 次高奖金 |
| 三等奖 | 前5+后0 | 10,000元 | 固定奖金 |
| 四等奖 | 前4+后2 | 3,000元 | 固定奖金 |
| 五等奖 | 前4+后1 | 300元 | 固定奖金 |
| 六等奖 | 前3+后2 或 前4+后0 | 200元 | 固定奖金 |
| 七等奖 | 前3+后1 或 前2+后2 | 100元 | 固定奖金 |
| 八等奖 | 前3+后0 或 前1+后2 或 前2+后1 或 前0+后2 | 15元 | 最低奖金 |

#### 双色球奖金等级

| 等级 | 中奖条件 | 奖金 | 说明 |
|------|----------|------|------|
| 一等奖 | 红6+蓝1 | 浮动 | 最高奖金 |
| 二等奖 | 红6+蓝0 | 浮动 | 次高奖金 |
| 三等奖 | 红5+蓝1 | 3,000元 | 固定奖金 |
| 四等奖 | 红5+蓝0 或 红4+蓝1 | 200元 | 固定奖金 |
| 五等奖 | 红4+蓝0 或 红3+蓝1 | 10元 | 固定奖金 |
| 六等奖 | 红2+蓝1 或 红1+蓝1 或 红0+蓝1 | 5元 | 最低奖金 |

## 功能 2: 号码生成器

### 2.1 随机生成

**生成规则：**
- 完全随机生成
- 号码不重复
- 自动排序（升序）
- 符合彩票规则

**生成算法：**
```
1. 根据彩票类型确定号码范围
2. 使用随机数生成器
3. 确保号码唯一性
4. 自动排序
5. 分别生成主号码和特殊号码
```

### 2.2 号码显示

**显示效果：**
- 主号码：蓝紫色渐变圆形
- 特殊号码：粉红色渐变圆形
- 中间用"+"号分隔
- 号码居中对齐

**视觉设计：**
- 圆形号码球
- 渐变色背景
- 阴影效果
- 响应式布局

### 2.3 复制功能

**复制格式：**
- 大乐透：`03, 12, 18, 25, 31 + 05, 09`
- 双色球：`05, 12, 18, 23, 28, 31 + 08`
- 福彩8：`08, 15, 23, 34, 45, 56, 67, 78`

**使用方法：**
1. 生成号码后
2. 点击"复制号码"按钮
3. 号码自动复制到剪贴板
4. 按钮显示"已复制"（2秒后恢复）
5. 可以粘贴到任何地方

## 使用场景

### 场景 1: 购买彩票前

**需求：** 想要一组随机号码

**操作：**
1. 打开号码生成器
2. 选择彩票类型
3. 生成号码
4. 复制号码
5. 去购买彩票

### 场景 2: 开奖后查询

**需求：** 想知道中了什么奖

**操作：**
1. 打开奖金计算器
2. 选择彩票类型
3. 上传彩票照片或手动输入
4. 查看中奖结果

### 场景 3: 批量生成号码

**需求：** 想要多组号码进行选择

**操作：**
1. 打开号码生成器
2. 多次点击生成
3. 记录喜欢的号码
4. 选择最满意的一组

## 注意事项

### 常见问题

**Q: 为什么每次生成的号码都不同？**
A: 号码是完全随机生成的，每次都会产生新的组合。

**Q: 可以指定某些号码吗？**
A: 目前不支持，但在未来计划中。

**Q: OCR 识别需要多长时间？**
A: 通常 2-3 秒，取决于网络速度和图片大小。

**Q: 支持哪些图片格式？**
A: 支持 JPG、PNG、WebP 等常见格式。

**Q: 可以保存生成的号码吗？**
A: 目前不支持，但可以复制到剪贴板或截图保存。
